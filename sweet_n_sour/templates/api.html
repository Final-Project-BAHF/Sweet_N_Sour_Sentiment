Sweet N' Sour Sentiment - API

The following API endpoints are available:



Get Transcripts: /api/get-scripts/<ticker>/<num-scripts> -- browses SeekingAlpha.com for the requested ticker and retrieves the number of Earnings Call Scripts specified in the 2nd parameter. The most recent n scripts will be retrieved.

Argument - <ticker> - the stock ticker symbol for the desired company
Argument - <num-scripts> - the number of most recent Earnings Call Transcripts to retrieve

For example: /api/get-scripts/GE/1

Returns - json with list of objects (one for each transcript requested) in the following format:
{
  'ticker': ticker symbol of company,
  'eps_info': text describing the EPS performance compared to analyst estimates,
  'revenue_info': text describing the Revenue performance compared to analyst estimates,
  'call_title': title of earnings call -- contains date and time of the call,
  'company_participants': [{
     'name': Participant Name,
     'company': Participant Employer,
     'affiliation': 'host' #indicates this participant is a member of the hosting company for the call
  }
  ...
  ],
  'call_participants': [{
     'name': Participant Name,
     'company': Participant Employer,
     'affiliation': 'guest' #indicates this participant is a guest attendee of the call
  }
  ...
  ],
  'paragraphs': [{
     'speaker': name of speaker,
     'content': paragraph of text from transcription for this given speaker,
     'call_section': specifies the section of the call in which this content was spoken ('presentation', 'operator_instruction', 'question', 'answer')
  }
  ...
  ]
}
